기본 선택자

태그, 클래스, 아이디, 전체 선택자

*{}
a{}
.클래스명 {}
#아이디명 {}

클래스(class) : 스타일이나 스크립트를 공통으로 적용시키기위해 해당 태그에 지정

아이디(id) : 하나의 html문서에서 각각의 요소를 중복되지 않게 아이디를 부여하여 각 요소를 구분하여 스타일이나 스크립트를 각각 적용하기 위해 태그에 지정

속성 선택자

태그의 속성에 따라 다르게 선택할 경우 사용

속성 존재 유무, 속성 일치, 속성 접두사, 속성 접미사, 포함하는 문자열

img[src]
img[src="mbc.png"]()
img[src|="문자"]{}
img[src^="문자"]{}
img[src$="문자"]{}
img[src*="문자"]{}
img[src~="문자"]{}

링크선택자

하이퍼링크 요소인 a요소의 현재 동작상태에 따라 달리 적용됨

a 요소에 href 속성의 값이 있는 경우 : link

a 요소에 href로 지정된 url이 방문한적이 있는 경우 : visited

요소에 마우스 포인터가 올라가는 경우 : hover

요소에 마우스를 누르고 있는 경우 : active

a:link{} a:visited {} a:hover{} a:active {}
위의 순서로 해야지 작동됨
